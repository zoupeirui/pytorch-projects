# 进阶与全链路 PyTorch 学习方案 (重构版)

## 全链路项目标准流程

每个阶段的 **Mini Project** 都必须打通以下 11 步端到端流程：
`数据集准备` ➔ `Dataset` ➔ `DataLoader` ➔ `模型搭建` ➔ `训练(Train)` ➔ `验证(Eval)` ➔ `保存Checkpoint` ➔ `加载Checkpoint` ➔ `推理预测` ➔ `可视化结果` ➔ `ONNX导出与部署建议`。

---

## 6 大实战阶段学习路线 (Roadmap)

### 第一阶段：视觉基础 (分类器起步)
- **目标**：掌握 PyTorch 完整训练管道的 11 个标准步骤。
- **项目**：**MNIST 手写数字分类** (经典 MLP 或简易 CNN)。
- **产出**：完成标准的全链路训练并导出 ONNX。

### 第二阶段：图像识别进阶 (自定义数据集)
- **目标**：掌握如何处理真实世界的不规则图像，进行数据增强（Data Augmentation）和使用预训练的视觉主干（如 ResNet）。
- **项目**：**猫狗分类 (Cat vs Dog)**。
- **产出**：学会使用迁移学习快速达到高准确率。

### 第三阶段：目标检测训练 (真实科研刚需)
- **目标**：理解并实践目标检测的核心思想（BBox 坐标回归与分类，IoU）。
- **项目**：**简易 YOLO 目标检测训练**（可选用极简版 YOLO 结构或现成的小型检测框架）。
- **产出**：可视化真实框与预测框的对比。

### 第四阶段：NLP 与注意力机制 (踏入大模型之门)
- **目标**：熟悉 Hugging Face transformers，掌握自注意力机制（Self-Attention）与文本数据的清洗和 Tokenization。
- **项目**：**Transformer 文本分类** (如 IMDB 电影评论情感分析，使用 BERT 或纯手写简易 Transformer)。
- **产出**：完成从文本到模型，再到情感分类结果的可视化。

### 第五阶段：多模态前置 (图文对齐)
- **目标**：学习对比学习（Contrastive Learning），理解如何将图像和文本映射到同一个向量空间。
- **项目**：**CLIP 模型微调 (Fine-tuning)**。
- **产出**：实现“以文搜图”或 Zero-shot 图像分类。

### 第六阶段：前沿多模态大模型 (VLM 微调实战)
- **目标**：掌握参数高效微调（PEFT / LoRA）在最新视觉大语言模型上的应用。
- **项目**：**LLaVA 或 Qwen-VL 微调 (LoRA)**。
- **产出**：通过自有数据微调，让多模态大语言模型拥有特定领域的“读图问答”能力，完全对标当前 AI 顶尖实验室的常规操作。
